# SpaceShip
一個用processing做的太空船小遊戲
## 玩法
- `w` 向上移動
- `a` 向左移動
- `s` 向下移動
- `d` 向右移動
- `space` 暫停/繼續
- `space`+左鍵 結束遊戲，回到主畫面
